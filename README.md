# Programming for Kids / 少儿编程

A gentle journey into the world of computers — programming concepts explained for kids.

走进计算机世界的启蒙之旅。

**[cs.johnsonlee.io](https://cs.johnsonlee.io/)**

## Chapters

1. Information Representation — How does a computer represent information?
2. Programming Language — How do we communicate with computers?
3. Data Structure & Algorithms *(coming soon)*
4. Software Engineering *(coming soon)*

## Language

Slides are available in English and Chinese. Switch languages with the toggle button in the top-right corner, or via the `?lang=` query parameter:

- English: [cs.johnsonlee.io/?lang=en](https://cs.johnsonlee.io/?lang=en)
- 中文: [cs.johnsonlee.io/?lang=zh](https://cs.johnsonlee.io/?lang=zh)

## Local Development

No build step required. Serve with any static HTTP server:

```bash
npx serve .
# or
python3 -m http.server
```

Then open `http://localhost:8000`.
